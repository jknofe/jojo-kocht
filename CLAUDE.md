# CLAUDE.md

This file provides guidance to Claude Code when working in this repository.

## What this repo is

A personal recipe collection (`jojo-kocht`) — markdown files originally exported from Google Keep. Language is German throughout.

## Structure

Each recipe or note is a standalone `.md` file in the repo root. There is no subdirectory structure.

- Recipes follow the pattern: `# Title`, optional italicised edit date, then `## Zutaten` and `## Zubereitung` sections.
- Non-recipe notes (shopping lists, how-tos) follow the same flat format but without the cooking sections.

## Conventions when adding or editing content

- Write in German.
- Keep the `## Zutaten` / `## Zubereitung` structure for recipes.
- Add new recipes as a new `.md` file and link it in `README.md` under the appropriate section (Rezepte or Sonstiges).
- File names are lowercase, hyphenated, descriptive (`my-recipe-name.md`).
- No frontmatter, no tags, no YAML — plain markdown only.
